## Setup project locally

1. Clone the repository

```
git clone git@github.com:isxwor/assessment.git
```

2. Install dependencies

```
# using yarn
yarn

# using npm
npm install
```

3. Run the development server

```
# using yarn
yarn start

# using npm
npm start
```
